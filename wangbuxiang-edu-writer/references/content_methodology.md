## 1. 选题方法论

### 选题交集模型

一个好选题 = 你的专业领域 + 读者的普遍兴趣 + 当下的时间节点

光是专业，可能曲高和寡。光是读者兴趣，可能流于肤浅。光是蹭热点，可能昙花一现。三者结合才能找到最精准的引爆点。

### 选题来源

**小红书**：大众舆论场里教育的话题焦点，重点关注评论区
**即刻**：教育博主、AI教育产品、行业洞察
**微博/抖音/B站**：热搜话题，至少两三个平台同时火才算真正"破圈"
**知乎**：教育话题下的高赞讨论，家长的深度困惑
**读者社群**：群友的难点、痛点、经常出现的需求，监测"有没有"、"推荐"等关键词

### HKR质检法

动笔前每个选题必须通过：
- **H (Happy)** 足够有趣、有悬念？标题和开头能让人好奇想点开？
- **K (Knowledge)** 有信息量？看完能学到新东西？
- **R (Resonance)** 能戳中情绪？让人"对对对我也这么想"？

S级选题三项兼备，及格至少两项，只占一项要重新审视，零项坚决放弃。

### 角色代入法（发散思考）

审视选题时代入不同角色：
- "一个很忙的普通家长"：能帮我省时间/解决具体麻烦吗？
- "一个爱学习的父母"：够有趣吗？能让我产生"卧槽还能这样？"的惊叹？
- "一个焦虑的陪跑者"：能让我学到新东西、有新角度吗？

## 2. 选题分类与过往爆款案例

### 按业务类型

**AI教育应用**（AI工具、学习方法、教育产品）
- 我用AI给孩子做了一个专属的学习助手。
- 豆包豆包，正在成为我家孩子的AI学习伙伴。
- 我用AI监控了孩子的学习状态，发现了一些意想不到的东西。
- AI陪娃写作业这两周，我好像卷入了一场新与旧的战争。

**教育理念**（育儿观念、成长陪伴、教育方法）
- 我花了三年才明白，孩子最需要的不是补课。
- 双减之后，我看到了两种完全不同的家庭。
- 那些"不焦虑"的家长，到底在想什么？

**泛教育文化**（深度观察、现象故事、观点解读、政策解读）
- 小学三年级，正在成为中国家长最焦虑的分水岭。
- 分析了100个孩子的成长轨迹后，我发现了一个残酷的真相。
- 那些"鸡娃"失败的家庭，都犯了同一个错误。

### 按内容类型

**产品评测+推荐**：实测体验为主，有明确好恶判断
**方法+教程分享**：手把手教读者操作，核心是具体步骤和案例
**现象观察类**：教育现象解读、家庭故事
**认知观点**：对事件的深度分析、教育理念分享
**AI实践**：自己动手做好玩的东西，展示AI在教育中的可能性

### 按写作原型（从8篇最佳文章中提炼）

**调查实验型**
亲自下场做一件事，边做边报道发现。最能建立信任感和叙事张力。
- 我用AI给孩子做了一个月学习规划 → 实际使用，看到效果变化
- 我花一周体验了所有AI学习工具 → 逐个测试、对比发现
- 我让孩子用AI写了一个月作文 → 注册账号、实际使用、一步步看变化
写法核心：过程叙事 + 层层发现 + 每一步都带真实反应

**产品体验型**
拿到产品实际使用，带着读者一起体验。不是评测报告，是"跟我一起玩"。
- 豆包豆包陪娃学习 → 从设置人设开始，逐个演示功能
- 这个AI学习工具 → 按玩法分类，每个玩法配真实case
写法核心：场景驱动 + 自己的真实感受 + 跟其他产品的自然对比

**现象解读型**
观察到一个有趣的现象，然后层层深入分析。
- 小学三年级分水岭 → 追问为什么这么关键 → 连接到认知发展 → 教育启示
- 有些孩子越管越叛逆 → 逐个案例分析 → 找到心理学原理 → 给出方法
写法核心：好奇心驱动 → 研究 → 知识掉落 → 哲学升维

**工具分享型**
分享一个实用的工具/方法，但必须用个人故事包裹，不能直接"推荐一个好用的xxx"。
- AI学习规划工具 → 从朋友的焦虑聊起 → 教育规划这个词的思考 → 工具展示 → 自己的测试结果
写法核心：个人故事铺垫 → 自然引出工具 → 自己的使用过程 → 效果展示

## 3. 内容节奏

节奏不是文笔问题，是一套让读者一路往下读的推进系统。

好的节奏像波动：每次围绕主线偏出去一点点（让读者喘口气、长见识、看案例），再用一句话拉回来继续推进。

**扣主线句**要高频出现——不需要长，一句就够，但频率要高。

**最忌讳**：突然偏离主线很远再硬拽回来，读者要花脑力"顺逻辑"，心流断掉。

节奏优先级：专业（底子）→ 观点（方向）→ 主线（绳子）→ 节奏（牵引力）

**对标参考**：凯叔讲故事、年糕妈妈、影视飓风、何同学、小Lin、九边

## 4. 创意案例工作法

### 三类锚点

**热点锚点**：当下最火的教育话题、政策变化、社会事件、网络热梗——流量最大公约数
**痛点锚点**：家长最高频、最头疼的问题——实用价值
**炫技点锚点**：AI工具本身最颠覆的核心功能——技术惊叹

### 包装成微型故事

1. 亮出"挑战"：想解决什么问题？
2. 展示"脑洞"：想到了什么骚操作？
3. 秀出"过程"：怎么一步步实现的？（截图、核心步骤）
4. 引爆"结果"：最终那个让人"卧槽"的作品

**准则**：案例必须有独立的观赏价值、创意价值和启发价值。如果自己都High不起来，坚决不发。
