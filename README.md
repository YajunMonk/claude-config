# claude-config

我的 Claude / Codex 写作与工作流技能配置仓库。

## 📝 内容创作工具

### khazix-writer
数字生命卡兹克公众号长文写作引擎。用卡兹克的风格撰写公众号文章，激发大家对 AI 的好奇。

### wangbuxiang-edu-writer
王不响公众号长文写作引擎。默认定位为「AI 实战朋友」，用于把 AI 学习、工作提效、生活应用、产品观察、未来教育支线、短视频脚本等素材整理或扩写成公众号文章。

### wangbuxiang-video-writer
王不响短视频口播写作引擎。用于把 AI 学习、工作提效、生活应用、AI 产品体验等素材整理成可直接录制的短视频逐字稿。

### hv-analysis
横纵分析法深度研究报告生成器。融合历时-共时分析、纵向-横截面研究设计，产出系统性研究报告。

## 🎨 排版工具

### wechat-layout-cloner
公众号排版组件级复刻器。从任意微信公众号文章中提取完整排版语言，生成可复用的 HTML 模板。

## 📥 安装方式

```bash
# 安装单个 skill
cd ~/.claude/skills
git clone https://github.com/YajunMonk/claude-config.git temp
cp -r temp/<skill-name> .
rm -rf temp
```
