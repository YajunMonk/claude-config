# Skills Collection

我的自定义 Claude Code Skills 合集。

## 📝 内容创作工具

### khazix-writer
数字生命卡兹克公众号长文写作引擎。用卡兹克的风格撰写公众号文章，激发大家对 AI 的好奇。

### wangbuxiang-edu-writer
王不响教育类公众号长文写作引擎。以「帮孩子建立未来能力」为使命的教育内容创作工具。

### hv-analysis
横纵分析法深度研究报告生成器。融合历时-共时分析、纵向-横截面研究设计，产出系统性研究报告。

## 🎨 排版工具

### wechat-layout-cloner
公众号排版组件级复刻器。从任意微信公众号文章中提取完整排版语言，生成可复用的 HTML 模板。

## 📥 安装方式

```bash
# 安装单个 skill
cd ~/.claude/skills
git clone https://github.com/YajunMonk/skills.git temp
cp -r temp/<skill-name> .
rm -rf temp
```
