# 赛博财经风格示例库

这份文档从历史稿件里提取典型段落、对比好坏写法、给出可直接模仿的模板。写稿卡壳时来查。

## 1. 开场 Hook 实战案例

### 反差钩（最常用，上手最快）
**样稿出处**：《AI 裁员剪辑稿》
> "有没有见过这样的公司，利润框框涨，老板却反手猛裁员，结果股价还涨了。"

拆解：三段式反差。A涨 → B反转 → C更反转。节奏像上楼梯，每句都让观众"？"一下。

**可模仿模板**：
- "有没有见过这样的事，XXX（正常逻辑），但XXX（反转一），结果XXX（再反转）。"
- "XXX（常识），但在 AI 行业，XXX（反常识）。"

### 荒诞钩
**样稿出处**：《一盒内存条，能买上海一套房》
> "一盒内存条，听起来像段子，但内存，真的在过去一年里，成了涨得最快的'硬资产'之一。"

拆解：标题级的荒诞事实 + "听起来像段子，但是真的" 这样的置信背书。

**可模仿模板**：
- "XXX，听起来像段子，但XXX是真的。"
- "XXX，可能比你想象的还要离谱。"

### 威胁钩
**样稿出处**：《一盒内存条》
> "即使是不用 AI 的普通人，也在给 AI 交税吗？"

拆解：直接把观众拽进事件，"你不用也得买单"。

**可模仿模板**：
- "你以为 XXX 跟你没关系？"
- "哪怕你不 XXX，你也在为 XXX 付钱。"

### 数据钩
**样稿出处**：《2026，美国AI没电了》
> "马斯克说：别光盯着芯片看，真正的决胜点在'能源'。"

拆解：大佬金句 + 反常识视角转移。

**可模仿模板**：
- "XXX（大佬）说：别盯着 XXX 看，真正的 XXX 是 XXX。"
- "XXX 的市值一夜蒸发 XXX 亿，但真正的问题不是 XXX。"

### 悬念钩
**样稿出处**：《黄仁勋访华》
> "黄仁勋这次访华第一天，在北京吃的就是这个馆子，点了这几样云南小菜。"

拆解：极日常的细节切入 + 暗藏玄机的氛围。观众会好奇"这跟财经有什么关系"。

**可模仿模板**：
- "XXX（大佬）干的第一件事，是 XXX（日常到反常识）。"
- "XXX 办公室里挂着一幅 XXX，今天这幅画有了新意思。"

## 2. 小标题怎么写

赛博财经的小标题不是信息概括，是**悬念 + 判断**。每一个小标题本身就是一个钩子，让观众愿意看下去。

### 反例 vs 正例

| 差（信息式） | 好（悬念+判断式） |
|---|---|
| 一、美国电力现状 | 一、硅谷的"核能狂想曲"与老旧电网 |
| 二、中国AI芯片发展 | 二、中国并不缺芯——华为产能与4000倍的赌注 |
| 三、DeepSeek最新模型 | 三、DeepSeek春节档预定，"降维打击" |
| 四、反英伟达联盟 | 四、硅谷内战，围剿英伟达 |
| 一、Block裁员 | 一、一封信引爆硅谷 |
| 二、股价分析 | 二、资本为什么为裁员欢呼？ |
| 三、AI能力边界 | 三、AI真的能干4000人的活吗？ |

### 小标题的几种公式

1. **戏剧化幕标题（最高阶，优先用）**：
   - 「第一幕：5:01 PM，硅谷最漫长的黄昏」（时间精确到分 + 金句场景）
   - 「第二幕：理想主义的'黄昏'与离职信」（情绪定性 + 关键事件）
   - 「第三幕：OpenAI 的'反水'与资本的冷血」（主角 + 带温度的定性词）
   - 「第四幕：谁才是真正的'造王者'？」（悬念问句 + 文化梗）
   - 公式：**第N幕 + [时间/情绪/主角] + 金句/反差/问句**
2. **判断式**：「硅谷的'核能狂想曲'」「资本欣赏这种果断」
3. **疑问式**：「AI 真的能干 4000 人的活吗？」「为啥普通内存也贵了」
4. **悬念式**：「一封信引爆硅谷」「硅谷内战，围剿英伟达」
5. **人物/地点+动词**：「Jack 的一封信」「黄仁勋的菜市场」
6. **编号+观点**（适合揭秘型稿件）：「一个误会，贵的到底是啥」「一个揭秘，AI 也缺内存吗」「一个追问——炒作还是结构性变化」

## 2.5 金句独立成段的 10 个范本

这是赛博财经最核心的节奏信号。写稿时要主动在下面这些位置埋一句金句单独成段：

**开场刚讲完事实后**
- "崩塌其实早有预兆。"
- "市场的情绪是非常诚实的。"

**数据暴击之后（让数据喘一口气）**
- "4000 人啊，朋友们。"
- "Anthropic 陷入了双输。"

**幕与幕过渡时**
- "然而，故事在这里出现了一个耐人寻味的转折。"
- "但在现在这个时间节点上，问题更加 tricky 了。"

**揭开反差时**
- "但这究竟是强强联手，还是引狼入室？"
- "资本无情但会算账。"

**升华前压轴一击**
- "所以，最后会消耗大量的水。"
- "这可能是英伟达在中国最艰难的一年。"

**结构规律**：金句独立成段最好 ≤ 20 字，干净利落，不叠加形容词，动词用硬的、名词用具象的。

## 2.6 戏剧化动词 + 画面定性短语库

**动词**（替换掉中立的"宣布/决定/发生"）：
引爆、围剿、拉入、死守、拆掉、反水、填补真空、拥抱、放逐、锁定、榨干、砸下、押注、掀桌、围堵、内战、断供、摊牌、投毒、游走、示弱、摁下暂停键

**画面定性短语**：
- 时间/氛围：最漫长的黄昏、理想主义的黄昏、72小时、Q1 最后一天、开年暴击
- 身份定性：造王者、灭霸、殉道者、养料、过路费、外围玩家、务实派、疯狂派
- 对决结构：红线、铁幕、柏林墙、黑暗森林、反xxx联盟、两种工业文明
- 金融隐喻：期货产品、硬通货、账面价值、战略储备、下一个万亿市场、屠杀机器

使用密度：**一期稿 8-15 个**。过少 → 像行业报告；过多（>15）→ 油腻尬演。

## 3. 双主播对话的正反例

### 反例：两个人轮流念稿（信息不推进）

```
🦊：Block 这次裁员了 4000 人，这是一件大事。
🐟：是的，裁员 4000 人确实是一件非常大的事，引起了市场的广泛关注。
🦊：市场反应很强烈，股价涨了很多。
🐟：股价上涨了 20% 多，市场反应的确很强烈。
```

问题：🐟只是在复述🦊。两个人说的是同一件事，信息完全没推进。

### 正例：互相接话 + 信息层层推进

```
🦊：Block 这家公司啊，就是前 Twitter 创始人 Jack Dorsey 搞的，美版支付宝 Cash App 就是他们做的。
🐟：上个月底发财报，毛利润同比涨了 24%。
🦊：但同一天，4000 多人收到离职通知，Jack 直接在推特上公布了全员信。
🐟：说了三件事儿。第一，裁员势在必行，赔偿我绝对给到位。
🦊：第二，裁员不是因为公司不行，是我们的 AI 实在太强了。与其小刀拉肉，不如撕创可贴。
🐟：第三，公司发展强劲，客户在涨，请大伙儿放心。
```

特点：🦊给背景和判断，🐟接着给数据和拆解。一句接一句，每一句都推进了新信息。节奏感像打乒乓球。

### 对话节奏的三种推进方式

**方式一：递进接话**（A说一半，B接下一半）
```
🦊：但为啥这些巨头都要自己去搞核能？
🐟：因为美国的公用电网，太老了！
🦊：老到什么程度？70% 的输电线路和变压器，年龄超过 25 年。
🐟：甚至是 50 年前建的。
```

**方式二：追问拆解**（B问，A答；或A问，B答）
```
🦊：资本市场已经用钱投票了。
🐟：投给谁？
🦊：投给李飞飞的"空间智能"路线。
```

**方式三：反转接话**（A讲完一个逻辑，B给一个反转）
```
🦊：很多人说 DeepSeek 这是因为买不到显卡才被逼出来的。
🐟：但其实这是一种更高等的进化。
```

## 4. 数据怎么给才有冲击力

### 原始数据 → 冲击性表达

| 原始 | 赛博财经风 |
|---|---|
| 微软和核电站合作 | 微软签了 **20 年**的协议，重启了"三里岛"核电站，锁定 **835 兆瓦**，这是美国历史上首次重启已退役核电站 |
| 华为芯片产能扩大 | 华为 Ascend 910C，**2026 年计划产量 60 万颗**。单卡不及 H100，但这个数量级面前，差距已被抹平 |
| 股票上涨很多 | 摩尔线程上市首日最高涨幅 **502%**，沐曦股份 **692.95%**，壁仞科技超 **110%** |
| Block 裁员后市场反应不错 | Block 裁员让市值增加约 **24 亿美元**，相当于每减掉一名员工，为公司创造约 **60 万美元**的账面价值 |
| 美国电网老旧 | 到 2040 年，光是要更换的输电线路就多达 **10 万英里**，相当于**绕赤道 4 圈**，需要超 **7000 亿美元**投资 |

### 数据表达的三个技巧

1. **对比转化**：10 万英里 → 绕赤道 4 圈；4-5 万一条内存 → 一盒 400-500 万 → 买一套上海 70 平的房
2. **单位人化**：裁员 4000 人 + 市值涨 24 亿 → 每减一人创造 60 万
3. **组合暴击**：不给一个数据，给 3 个同方向数据连击（502%、692%、110% 三连）

## 5. 画面/报告提示的规范（极简版）

剪辑的细节由后期老师把控。写稿人只负责标两件事：**需要上画面的位置**和**需要上报告/文件的位置**。

### 只用两种标注

```
[画面:xxx]   需要上一张具体的图/素材/视觉画面
[报告:xxx]   需要露出报告原文/文件截图/数据页
```

**不要再写**：`【压字:xxx】`、`【人名条:xxx】`、`【划重点:xxx】`、`【素材:xxx】`、`【数据图:xxx】`、`【录屏:xxx】`、`【镜头:xxx】`、`【转场:xxx】`、`【b-roll:xxx】`——这些都是后期老师的活，写稿时标了反而添乱。

### 好的标注（具体、可执行）

```
🦊：2.7%。这是斯坦福大学2026 AI Index报告里,中美顶级大模型的性能差距。[画面:斯坦福2026 AI Index报告封面]

🐟:这份报告是4月7日发布的。[画面:报告发布页面或时间节点]

🐟:结论就一句话——2025年1月,中美顶级模型的能力差距从17.5%收窄到2.7%。[报告:Chatbot Arena对比数据那一页]

🦊:黄仁勋这次访华第一天,在北京吃的就是这个馆子。[画面:黄仁勋在北京餐馆用餐的街拍/现场照]
```

### 差的标注（模糊或过度细化）

```
[画面:配个画面]                        // 太模糊,后期不知道找什么
[镜头:黄仁勋特写,推镜]                 // 过度,机位是后期的活
[压字:2.7%,红色加粗,第3秒出现]         // 过度,字幕是后期的活
[b-roll:硅谷鸟瞰 + 数据中心 + 核电站]   // 过度,b-roll 是后期的活
```

### 密度参考

- `[画面:xxx]`：一期稿 5-10 处。集中在重要人物首次出现、关键地点/现场、涉及具体产品/实物（内存条、核电站、芯片）、需要视觉冲击力的反差时刻
- `[报告:xxx]`：一期稿 0-3 处。只在稿子真的在讲某份报告/财报/研报/政策文件时才标
- **不要句句标**。口播以语言为主,画面是在真正需要"这里得上图才不抽象"的地方才出现

## 6. 升华收尾的几种写法

### 写法一：双极对决式
**《2026，美国AI没电了》**
> "看懂了吗？2026 年的算力世界，已经彻底分裂。
>
> 美国选择了'重资源'+'单点爆发'。
> 中国选择了'高效率'+'网络协同'。
>
> 这不仅是技术的博弈，这是两种工业文明的对决。"

### 写法二：历史参照式
**《黄仁勋访华》**
> "虽然黄仁勋开年就用脚投票了，但今年的剧本，不再由硅谷单独撰写，太平洋对岸的这片市场，还有多少，会再次选择英伟达？"

### 写法三：哲学追问式
**《硅谷72小时》**
> "这一轮技术进步，到底是不是一个奥本海默时刻？
>
> AI 还没给出答案，但留给人类思考的时间，已经不多了。"

### 写法四：结构闭环式
**《一盒内存条》**
> "所以，今天内存为什么这么贵？不是因为内存突然变值钱了，而是因为 AI 正在重写资源的优先级。
>
> 当一项技术开始和普通人抢同一套工业体系的时候，价格变化，只是最先被你感知到的那一部分。"

### 写法五：未来悬念式
**《AI 裁员剪辑稿》**
> "几次工业革命下来，机器都代替了一些岗位，也创造了新的岗位。但这次，真的会是一样吗？
>
> 毕竟 AI 取代的，已经不仅是重复性的体力，而是我们引以为傲的'智能'本身。"

### 通用结构
```
【回到现象】用一句话点回整期在讲什么
 ↓
【拉高视野】从具体事件升到文明/历史/产业的大坐标
 ↓
【抛出余韵】留一个悬念/问题/判断，不给完整答案
 ↓
【品牌尾】这里是赛博财经，点个关注...
```

### 禁忌

- 不要用"总之""综上所述""让我们拭目以待""未来可期"这种空话
- 不要自我总结"今天我们聊了xxx，主要讲了三点"
- 不要反鸡汤化"无论如何，勇敢面对未来"
- 升华要跟本期具体事件严丝合缝，不是空降一个大词

## 7. 典型金句库（可借用改写）

按场景分类，写稿时查缺补漏：

**产业博弈**
- "这不仅是技术的博弈，这是两种工业文明的对决。"
- "你封锁我的硬件上限，我就用软件把硬件的潜力榨干。"
- "硅谷内战已经动手了。"

**资本冷酷**
- "人的数量，正在从资产，变成需要被优化的累赘。"
- "资本无情但会算账。"
- "每家公司的个体决策都是理性的，集体的结果是灾难性的。"

**普通人锚点**
- "即使是不用 AI 的普通人，也在给 AI 交税。"
- "当一项技术开始和普通人抢同一套工业体系的时候，价格变化，只是最先被你感知到的那一部分。"

**技术哲学**
- "把 AI 从'描述世界'推进到'模拟世界'。"
- "AI 取代的，已经不仅是重复性的体力，而是我们引以为傲的'智能'本身。"

**时代坐标**
- "2026 年的算力世界，已经彻底分裂。"
- "这场比赛才刚刚开始。"
- "这可能是英伟达在中国最艰难的一年。"
